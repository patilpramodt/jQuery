Task 21 - Cloning same address

  • When user fills something in current address controls and click on "same as current address" then current address control data should get duplicated in billing address controls
  • If "Same as current address" check box is checked then billing address controls should be in read only mode and not editable
  • If "Same as current address" check box is not checked then billing address controls should be editable
  • If "Same as current address" check box is checked and user makes any changes in current address should get changed in billing address as well.
